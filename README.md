# Projet_math_app
Application graphique de résolution et de modélisation mathématique


Ce projet s’inscrit dans le cadre de développement Python scientifique. L’objectif est de concevoir une application graphique permettant de résoudre et modéliser des problèmes mathématiques dans quatre domaines : systèmes linéaires, programmation linéaire, régression linéaire et processus stochastiques. L’interface est développée avec PyQt5, et les calculs sont réalisés à l’aide de bibliothèques comme NumPy, Matplotlib, PuLP et Scikit-learn.
